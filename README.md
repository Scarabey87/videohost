# AI Video Platform

Платформа для просмотра видео с AI-персонами с VIP-подпиской и админ-панелью.

## Возможности

### Для посетителей:
- Просмотр приветственного экрана с описанием платформы
- Сетка популярных видео
- Карточки доступных персон (AI-аватаров)
- Ограниченный просмотр контента без регистрации

### Регистрация и вход:
- Уникальное имя пользователя и email
- Пароль от 8 символов
- Защита от подбора (блокировка после 5 попыток)
- Автоматический вход после регистрации

### Для пользователей:
- Просмотр каталога видео с фильтрами
- Просмотр видео с плеером (play/pause, громкость, fullscreen)
- История просмотров
- Редактирование профиля (пароль, email, аватар)
- Оформление VIP-подписки (7, 30, 90 дней)

### VIP-функции:
- Доступ ко всем премиум-видео
- Размытие превью VIP-видео для не-VIP пользователей
- Значок замка 🔒 на VIP-контенте

### Для администраторов:
- Загрузка видео (до 100 МБ, форматы: MP4, WebM, AVI, MOV, MKV)
- Управление пользователями (поиск, фильтры, редактирование, удаление)
- Управление персонами (добавление, редактирование, удаление)
- Управление видео (публикация, VIP-статус, массовые действия)
- Просмотр логов действий
- Настройки сайта

## Безопасность:
- Хеширование паролей
- Защита от CSRF/XSS
- Rate limiting
- HTTPS (в production)
- Защита от clickjacking

## Установка

```bash
pip install flask flask-sqlalchemy flask-login werkzeug
python app.py
```

## Структура проекта:
```
/workspace
├── app.py              # Основной файл приложения
├── templates/          # HTML шаблоны
│   ├── base.html       # Базовый шаблон
│   ├── index.html      # Главная страница
│   ├── login.html      # Вход
│   ├── register.html   # Регистрация
│   ├── videos.html     # Каталог видео
│   ├── watch.html      # Просмотр видео
│   ├── personas.html   # Список персон
│   ├── persona_detail.html  # Страница персоны
│   ├── vip.html        # VIP-подписка
│   ├── profile.html    # Профиль пользователя
│   ├── edit_profile.html  # Редактирование профиля
│   ├── admin/          # Админ-панель
│   └── errors/         # Страницы ошибок
├── static/             # Статические файлы
│   ├── css/            # Стили
│   └── js/             # JavaScript
└── uploads/            # Загруженные файлы
    ├── videos/         # Видеофайлы
    ├── previews/       # Превью
    ├── avatars/        # Аватары
    └── personas/       # Фото персон
```

## Технологии:
- Flask (веб-фреймворк)
- SQLAlchemy (ORM)
- Flask-Login (аутентификация)
- SQLite (база данных)
- iOS-style дизайн с тёмной/светлой темой

- Laptop (1366×768+)
